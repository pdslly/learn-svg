SVG line元素使用

什么是line元素

SVG line元素用于在SVG中绘制线条

SVG line示例

        <svg xmlns="http://www.w3.org/2000/svg" style="border: 1px red dotted;" xmlns:xlink="http://www.w3.org/1999/xlink" height="300px" width="300px" style="border: 1px dotted red;">
             <line x1="0" y1="0" x2="300" y2="300" style="stroke: #006600;"/>
             <line x1="0" y1="300" x2="300" y2="0" style="stroke: #006600;"/>
        </svg>
初始点是属性设置的(x1, y1)，结束点是属性设置的(x2, y2)。
