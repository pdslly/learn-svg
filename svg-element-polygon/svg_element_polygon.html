SVG polygon元素使用

什么是polygon元素

SVG polygon元素用于在SVG中绘制多边形（三边或者更多）

SVG polygon示例

        <svg xmlns="http://www.w3.org/2000/svg" style="border: 1px red dotted;" xmlns:xlink="http://www.w3.org/1999/xlink" height="300px" width="300px" style="border: 1px dotted red;">
             <polygon points="10,10 290,10 150,290" style="stroke: red; stroke-width: 2; fill: #006600;"/>
        </svg>
你可能注意到了，即使只列出了三个点，三边也都会被描边。 这是因为 polygon 元素是在所有点之间绘制线条，包括结束点到初始点之间。而 polyline 不会，这大概是 polyline 与 polygon 之间唯一的区别。 
        <svg xmlns="http://www.w3.org/2000/svg" style="border: 1px red dotted;" xmlns:xlink="http://www.w3.org/1999/xlink" height="300px" width="300px" style="border: 1px dotted red;">
             <polygon points="10,10 290,10 290,290 150,150 10,290" style="stroke: red; stroke-width: 2; fill: #006600;"/>
        </svg>
